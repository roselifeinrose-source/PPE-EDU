Project: Git Workflow Agent Prompt

Description:
A reusable AI-agent workflow specification that automates:

* Repository synchronization
* Default branch detection (main/master)
* Automatic DEV branch incrementation
* Git stash management
* Conventional commit generation
* Commit validation
* Branch creation and push workflow
